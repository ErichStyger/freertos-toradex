Clone of the Toradex i.MX7 repository for the Colibri modules:
git://git.toradex.com/freertos-toradex.git


Extra files:
gcc_env.bat:
Run this batch file on Windows to setup the GCC environment variables. This allows to set the paths without the needs to change the system paths.
Update the paths in the gcc_env.bat and point it to your cmake, mingw and GNU toolchain
